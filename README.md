# BabaileOgul4Yas

Bu proje tek sayfalik, statik bir "random oyun fikri" sitesidir.

## Yerel acma

`index.html` dosyasini tarayicida acman yeterli.

## Cloudflare Pages ile yayin

1. GitHub'da yeni repo ac (ornek: `babaileogul4yas`)
2. Bu klasordeki dosyalari repoya yukle
3. Cloudflare Pages > Create Project > Connect to Git sec
4. Repo secimi yap
5. Build command bos birak, output directory `/`
6. Subdomain olarak `babaileogul4yas` yazip kaydet

Not: `babaileogul4yas.pages.dev` musaitse atanir, musait degilse benzer bir ad secmen gerekir.
